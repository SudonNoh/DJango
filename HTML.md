
<h1>HTML 공부</h1>

```html
<!-- 제목 -->
<h1></h1>
<h2></h2>
<h6></h6>
```

```html
<!-- 단락 -->
<p></p>
```

```html
<!-- 줄바꿈 -->
<br>
```

```html
<!-- 텍스트 서식 미리 정의 -->
<pre></pre>

<!-- 예시 -->
<pre>
  Sudonnoh 의
  GitHub
</pre>
```

<pre>
  Sudonnoh 의
  GitHub
</pre>

```html
<!-- 수평 가로 구분선 -->
<hr>

<!-- 예시 -->
<hr>
<p>미리보기<p>
<hr>
```
<hr>
<p>미리보기<p>
<hr>

```html
<!-- 굵은 글씨 -->
<b></b>
<strong></strong>

<!-- 예시 -->
<p><b>굵은 글씨</b> 입니다.<p>
<p><strong>굵은 글씨 강조</strong> 입니다.<p>
```
<p><b>굵은 글씨</b> 입니다.<p>
<p><strong>굵은 글씨 강조</strong> 입니다.<p>
  
```html
<!-- 이탤릭체 -->
<i></i>
<em></en>

<!-- 예시 -->
<p><i>이탤릭체</i> 입니다.<p>
<p><em>이탤릭체 강조</em> 입니다.<p>
```
<p><i>이탤릭체</i> 입니다.<p>
<p><em>이탤릭체 강조</em> 입니다.<p>

```html
<!-- 하이라이팅 효과 -->
<mark></mark>

<!-- 예시 -->
<p><mark>하이라이팅 효과</mark> 입니다.<p>
```

```html
<!-- 삭제효과 -->
<del></del>

<!-- 예시 -->
<del>"이부분"</del>을 삭제합니다.
```
<p><del>"이부분"</del>을 삭제합니다.</p>

```html
<!-- 삽입 효과 -->
<ins></ins>

<!-- 예시 -->
<ins>"밑줄 친 부분"</ins>입니다.
```
<p><ins>"밑줄 친 부분"</ins>입니다.</p>
